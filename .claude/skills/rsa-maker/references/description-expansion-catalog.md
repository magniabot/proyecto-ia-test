# Description Expansion Catalog — Reference

Read during description composition. Provides single-angle and combined-angle patterns with examples.

## Core Constraints

- **Max:** 90 characters
- **Recommended:** 75-90 characters
- **Non-duplication rule:** Descriptions add new information beyond headlines
- **Query bolding:** Include core keyword naturally in at least one description — Google bolds matching text

---

## B. Combined-Angle Patterns

Stack 2-3 angles into one description for maximum density.

### Pattern 1: Problem + Solution

`[Pain acknowledgment] + [Your answer]`

| Example | Chars |
|---------|-------|
| "Tired of deals slipping through the cracks? Our CRM keeps your entire pipeline organized." | 90 |
| "Showroom markup is ridiculous. Get the same designer quality at factory-direct prices." | 87 |
| "Wasting budget on clicks that don't convert? Our SaaS-focused approach changes everything." | 90 |
